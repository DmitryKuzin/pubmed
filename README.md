HOW TO START THIS PROJECT : 

    LINUX/MAC OS:
    
    1. Give permisions to execute .sh files
        
        chmod +x gradlew.sh && 
        chmod +x initializeElasticSearch.sh && 
        chmod +x start.sh
       
    2. run elasticsearch
        ./elasticsearch-6.2.3/bin/elasticsearch
    
    Fitst run :
        run sh scripts
        ./initializeElasticSearch.sh &&
        ./start.sh
    
    Other runs :
        run sh script
        ./start.sh
